## Requirements
- Python 3.8x +
- Pandas 2.0.0
- NumPy 1.24.2

- To find out the version of Pandas and NumPy, simply run (replace `pip` with `conda` where appropriate):

```bash
$ pip3 list | grep -w "pandas\|numpy"
numpy             1.24.2
pandas            2.0.0
```
